### 1. Para instalar

#### LINUX

```console
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

#### WINDOWS

```console
python3 -m venv .venv
source .venv/Scripts/activate
pip install -r requirements.txt
```

### 2. Para executar

```console
export DEBUG=True
export DEVELOPMENT_MODE=True
python manage.py runserver
```
